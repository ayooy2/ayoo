🔧 待办：图片存储迁移到 R2

当前状态：图片以 base64 存储在 D1 中，单列上限 256KB（约 180KB 原图），免费计划 5GB 总容量。

迁移步骤：

  1. 登录 Cloudflare Dashboard
  2. 左侧菜单点击 R2 Object Storage（需要绑定支付方式，免费额度 10GB）
  3. 点击 Create bucket，名称输入 ayoo-images，位置选 Automatic
  4. 进入 bucket → Settings → Public Access → 启用 R2.dev subdomain
  5. 编辑 wrangler.toml 添加：
[[r2_buckets]]
binding = "IMAGES"
bucket_name = "ayoo-images"
  1. 修改 functions/api/images.js：POST 写入 R2（env.IMAGES.put()），GET 从 R2 读取（env.IMAGES.get()），D1 只存元数据
  2. 创建迁移文件：将 images 表的 data TEXT 改为 r2_key TEXT
  3. 运行迁移：npx wrangler d1 migrations apply a-site-db --remote
  4. 部署：npx wrangler pages deploy public

迁移后：图片大小限制放开到 5MB，免费 10GB 存储，无出口费用。现有 D1 中的图片需要写脚本迁移到 R2。
